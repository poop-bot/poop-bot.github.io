Command Help

!help - Get Help On Commands. Usage - !help

!kick - Kick members from the server. Usage - !kick @member reason

!ban - Ban members from the server. Usage - !ban @member reason

!meme - Get a random meme. Usage - !meme

!rps - Play "Rock, Paper, Scissors" with me. Usage - !rps choice ("r", "p" or "s")

!wish - Wish a member something. Usage - !wish @member wish

!addrole - Add a role to a member. Usage - !addrole @member @role

!removerole - Remove a role from a member. Usage - !removerole @member @role

!share - Get links to Poop Bot. Usage - !share
